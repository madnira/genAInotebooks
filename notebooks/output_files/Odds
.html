Several ways exist to print odd numbers from 1 to 50 in C. Here are two examples: One using a `for` loop and another using a `while` loop. **Method 1: Using a `for` loop** This method is generally preferred for its conciseness and readability. ```c #include int main() { // Loop through numbers 1 to 50, incrementing by 2 to get only odd numbers. for (int i = 1; i <= 50; i += 2) { printf("%d ", i); } printf("\n"); // Add a newline at the end for better formatting return 0; } ``` **Method 2: Using a `while` loop** This method demonstrates a slightly different approach, but achieves the same result. ```c #include int main() { int i = 1; while (i <= 50) { printf("%d ", i); i += 2; // Increment by 2 to get the next odd number } printf("\n"); // Add a newline at the end for better formatting return 0; } ``` Both programs will produce the same output: a sequence of odd numbers from 1 to 49 separated by spaces. The `for` loop version is generally considered more efficient and easier to read for this specific task. Choose whichever method you find more understandable. Remember to compile and run the code using a C compiler (like GCC) before executing.
