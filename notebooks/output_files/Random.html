Python offers several ways to generate random numbers. Here are a few examples, demonstrating different levels of control and use cases: **1. Generating random integers within a range:** This is the most common use case. We use the `random.randint()` function. ```python import random def generate_random_integer(min_val, max_val): """Generates a random integer between min_val (inclusive) and max_val (inclusive).""" try: min_val = int(min_val) max_val = int(max_val) if min_val > max_val: raise ValueError("Minimum value cannot be greater than maximum value.") return random.randint(min_val, max_val) except ValueError as e: return f"Error: {e}" # Example usage: min_num = 1 max_num = 100 random_number = generate_random_integer(min_num, max_num) print(f"Generated random integer between {min_num} and {max_num}: {random_number}") #Example of error handling random_number = generate_random_integer(100,1) print(random_number) random_number = generate_random_integer("a",10) print(random_number) ``` **2. Generating random floating-point numbers:** Use `random.random()` for a number between 0.0 (inclusive) and 1.0 (exclusive), or `random.uniform()` for a number within a specified range. ```python import random def generate_random_float(min_val=0.0, max_val=1.0): """Generates a random floating-point number between min_val (inclusive) and max_val (exclusive).""" try: min_val = float(min_val) max_val = float(max_val) if min_val > max_val: raise ValueError("Minimum value cannot be greater than maximum value.") return random.uniform(min_val, max_val) except ValueError as e: return f"Error: {e}" # Example usage: random_float = generate_random_float() print(f"Generated random float between 0.0 and 1.0: {random_float}") random_float = generate_random_float(2.5, 7.8) print(f"Generated random float between 2.5 and 7.8: {random_float}") random_float = generate_random_float(7.8,2.5) print(random_float) ``` **3. Seeding the random number generator:** For reproducible results, you can seed the random number generator using `random.seed()`. The same seed will always produce the same sequence of "random" numbers. ```python import random random.seed(42) # Setting the seed to 42 random_number1 = random.randint(1, 10) random_number2 = random.randint(1, 10) print(f"Random number 1: {random_number1}") print(f"Random number 2: {random_number2}") random.seed(42) # Setting the seed again to 42 random_number3 = random.randint(1, 10) random_number4 = random.randint(1, 10) print(f"Random number 3: {random_number3}") print(f"Random number 4: {random_number4}") #Same as random_number1 ``` Remember to `import random` at the beginning of your script to use these functions. These examples include basic error handling to make them more robust. For more advanced random number generation (e.g., different distributions), consider using the `numpy.random` module, which offers a wider array of functionalities.
